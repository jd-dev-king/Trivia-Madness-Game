FEATURE SET

Built as a complete desktop experience

01

Live trivia API

Downloads real multiple-choice questions from Open Trivia DB and safely decodes special characters.

02

Configurable gameplay

Players select multiple categories, difficulty, question count, and time allowed for every question.

03

Visual feedback

Rounded answer controls, color-coded results, and a timer bar that changes as time runs out.

04

Persistent leaderboard

SQLite stores player scores, percentages, categories, difficulty, and completion timestamps.

05

Reliable desktop workflow

Background API loading keeps the interface responsive, while errors are surfaced through clear dialogs.

06

Reusable UI system

A centralized theme and reusable rounded Swing components create a consistent visual identity.
